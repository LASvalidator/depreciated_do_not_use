lasvalidate
===========

An open source implementation of the ASPRS LAS Validation Suite (LVS) for a speedy and transparent implementation that provides interested parties with the opportunity for bug reports, design concerns, and other input at an early stage.

Temporary Restriction: This code is currently being used by rapidlasso GmbH for bidding on the request for proposals (RFP) that was issued by the ASPRS for the implementation of the LAS Validation Suite (LVS). It is forbidden to make use of any parts of this code to submit a competing bid.
